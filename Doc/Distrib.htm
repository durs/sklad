Описание дистрибутива "СКЛАД"

Данный дистрибутив представляет НЕ демонстрационную, полнофункциональную версию программы НЕ ограниченную по времени и ни каким другим способом! Правда при этом я не гарантирую технической поддержки и своевременных ответов на Ваши вопросы. Однако, если они появятся, то смело пишити письмо, я постораюсь на них ответить

Дистрибутив состоит из архива sklad_3.zip, в котором находятся следующие файлы:

Sklad \ Sklad.exe - главный исполняемый модуль программы.

Sklad \ Sklad.cfg - файл настроек программы.

Sklad \ Queries.xml - файл описания запросов для выполнения в разделе "Анализ".

Sklad \ Backgrnd.bmp - фоновый рисунок для панели просмотра.

Sklad \ Changes.txt - текстовой файл, содержащий информацию об изменениях в текущей версии программы.

Sklad \ Udf \ dy_udf.dll - библиотека расширенных функция для InterBase Server, этот файл необходимо поместить одноимённую папку из каталога установки Interbase (например сюда: C:\ Program files \ InrerBase \ Udf).

Sklad \ Udf \ dy_udf.sql - описание SQL инструкций по использованию функций в библиотеке dy_udf.dll.

Sklad \ Database \ Empty.gdb - шаблон базы данных для хранения документации склада

Sklad \ Database \ Сладкоежка.gbk - архив базы данных кондитерского склада "Сладкоежка", которая приводится в качестве примера использования программы "Склад". Для её разархивирования необходимо использовать утилиту gbak.exe которая постовляется вместе с сервером Interbase или через меню программы "Склад" : сервис -> утилиты -> инструменты администратора

Sklad \ Database \ Сладкоежка.gdb - тоже самое, только в разархивированном виде, в комплет обычно входит один из файлов

Sklad \ Doc \ *.* - документация, к сожалению полной документации по работе с программой пока нет поэтому если что непонятно пишите по адресам указанным ниже.

Sklad \ Script \ *.* - скрипты, для обновления структуры баз данных старых версий.

Пожелания, замечания и вопросы можно направлять по адресам: dy@hotbox.ru, durs@ntcnvg.ru
