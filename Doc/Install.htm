Порядок установки программы "Склад"

  • Разархивируйте дистрибутив программы Sklad_3.zip ( ~4 Mb) в отдельную папку (Например c:\Sklad ). Если у Вас установлена программа WinZip, то это можно сделать нажав правой кнопкой мыши на архиве и выбрав соответствующий пункт меню.
  • Установите сервер Interbase 6.0 (или FireBird 1.0 или ещё какой клон Interbase, см. здесь). на ваш компьютер или на сервер вашей организации. Если сервер устанавливается на отделный компьютер (т.е. сервер), то на каждую клиенскую машину необходимо дополнительно устанавливать Interbase Client, с помощью той же программы инсталляции (необходимо только убрать галочку "Interbase Server", что бы небыло лишнего мусора на вашем компьютере). Не забудьте, что в этом случае база данных так же должна находиться на сервере, если что то не получится смотрите документацию по Interbase, её можно найти здесь: http://www.ibase.ru.
  • Перепишите библиотеку dy_udf.dll из папки Udf дистрибутива Sklad в одноимённую папку сервера Interbase (например сюда в C:\Program files\InterBase\Udf, всё зависит какую папку вы указали при инсталляции Interbase сервера).
  • Запустите на выполнение программу Sklad.exe. По умолчанию в состав дистрибутива включена база данных с примером при подключении к которой будет запрошено имя пользователя и пароль для доступа, если вы установили Interbase только что, то User name = "SYSDBA", а password = "masterkey". С помощью меню "Сервис" -> "Утилиты" -> "Инструменты администратора" Вы сможете cоздайте новую базу данных и тогда можно приступать к работе :).
